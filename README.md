# Trabalho-Final--Algoritmo-e-Programa-o---A3

Descrição do Projeto

O projeto consiste em um Sistema de Gerenciamento de Pedidos para Restaurante, desenvolvido na linguagem Java. O sistema permite o cadastro e autenticação de clientes, visualização do cardápio, realização de pedidos, gerenciamento do carrinho de compras e finalização de pedidos.

Como Executar o Sistema
Baixe ou clone o repositório do projeto.
Abra o projeto em uma IDE Java, como IntelliJ IDEA ou Eclipse.
Compile os arquivos do projeto.
Execute a classe Main.
Utilize o menu exibido no terminal para acessar as funcionalidades do sistema.
